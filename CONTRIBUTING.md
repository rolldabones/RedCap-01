# CONTRIBUTING

Thank you for your interest in RedCap-01.

## Purpose

Contributions should strengthen the repository's single governing question:

> Does Enterprise Risk Management improve the decisions that determine whether organizational objectives are achieved?

Avoid expanding the repository into a general ERM framework.

## Principles

Contributions should be:

- evidence-oriented
- practical
- concise
- objective-first
- decision-led
- framework-neutral where practical

## Preferred contributions

- Examples
- Templates
- Worked assessments
- Editorial improvements
- Research references
- Implementation experience

## Please avoid

- duplicating existing GRC next™ repositories
- unnecessary complexity
- unsupported claims
- excessive scoring systems
- documentation for its own sake

Note that `RedCap-01-Custom-GPT.md` mirrors the deployed Custom GPT verbatim. Proposed changes to the instruction block are welcome as suggestions but are merged only together with a production update.

## Pull requests

Please explain:

- the problem addressed
- why the change improves decision usefulness
- whether the change is backward compatible
- supporting references where appropriate
